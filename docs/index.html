Features

Everything you need in a shared list

Built on top of Mattermost, so your shopping list is collaborative, real-time, and accessible from any client.

🔗

Mattermost-backed

Your list lives in a Mattermost channel. Anyone with access can add items from any Mattermost client or the app.

👆

Swipe to cart

Swipe items left or right to toggle them in and out of your cart. One-handed, fast, satisfying.

✅

Batch checkout

When you're done shopping, tap Checkout to clear all carted items at once with a Mattermost reaction.

💾

Offline cache

Items are cached locally with SQLite. The list loads instantly, even without a network connection.

🌙

Light & dark themes

Choose between automatic, light, or dark mode. The app follows your system preference by default.

🔄

Pull to refresh

Drag down to instantly sync the latest items from your Mattermost server.

Get started

Up and running in minutes

All you need is Flutter and a Mattermost instance with a dedicated channel.

Clone & install

Grab the repo and fetch dependencies.

git clone … && flutter pub get

Sign in

Launch the app and sign in with your Mattermost server URL, username, and password.

Pick a channel

Open Settings and select the channel you want to use as your shopping list. You're all set!
